What this is!

This is my Continous Integration and Continous Deployment Server wich basically means, that this server will integrate, install, manage and deploy every current active and soon active project for me automatically! With just 5 - 10 lines of Code! Just upon either an git commit or an manual command to this services backrunning nodejs process! This can be fulfilled cause its running my Command-Manager (CLI Library). So i can setup such of the down listed commands!

What this website will be!

This website should soon show a list of all the recent builds and especially for me I want to implement some kind of managing and starting the tasks! Maybe build some kind of UI like Jenkins but in theme of Bootstrap!

Commands

  • trigger:   Triggers a specific deployment process

  • list:   Lists all currently registered deployment processes

  • history:   Lists a history of all the already deployed processes and their statusses

  • info:   Shows Informations about one specified deployment process
